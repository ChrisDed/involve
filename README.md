Instructions to install and run backend environment on Python 3.9:
In the terminal:
* cd back-end
* python3.9 -m venv env
* source env/bin/activate
* Pip3 install -r requirements.txt
* unvicorn main:app

Instructions to setup front-end react app:
In the terminal:
* cd front-end/r2d2/
* npm install
* npm run dev
